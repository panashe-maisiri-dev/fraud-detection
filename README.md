# 💳 End-to-End Banking Fraud Detection & Analytics Platform

This is an end-to-end Python project built to develop a fraud detection mechanism for banking transactions. It also involves creating an analytics tool used to analyze data and provide initial insights before further complex analysis is done.

---

## 📌 Project Phases

1. **Initial EDA of the Raw Fraud Data** — Data cleaning, exploratory data analysis, and numerical insights on raw transaction records. *(Current Phase)*
2. **Building of Analytics Tool** — Developing an interactive dashboard to explore key transaction metrics and visualize data patterns dynamically.
3. **Model Development & Deployment** — Building, evaluating, and integrating predictive fraud detection models into the platform.

---

## 🛠️ Tech Stack
* **Language:** Python
* **Environment:** VS Code & Jupyter Notebooks
* **Analysis & Visualization:** Pandas, NumPy, Matplotlib, Seaborn
* **Version Control:** Git & GitHub
